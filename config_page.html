Your Awesome Entertainment Hub Addon Configuration

This addon is designed for **self-hosting** and relies on environment variables for API keys. It does not require in-addon configuration through this page for its core functionality (TMDb, Trakt).

For streaming, please ensure you have the **Torrentio addon** installed in Stremio and that it is configured with your **Real-Debrid** account.

How to Install and Configure (Self-Hosted)

1. **Clone this project** from your Git repository to your Debian 12 server.

2. **Create a .env file** in the root of the project (next to app.py) and add your API keys:

TMDB_API_KEY=your_tmdb_api_key_here
TRAKT_CLIENT_ID=your_trakt_client_id_here
TRAKT_CLIENT_SECRET=your_trakt_client_secret_here

3. **Install Python dependencies** from requirements.txt within a virtual environment.

4. **Set up Gunicorn and Nginx** as a reverse proxy to serve the Flask application.

5. **Access your addon** in Stremio by its URL (e.g., http://your-server-ip/manifest.json).

Refer to the README.md file in the project's Git repository for detailed setup instructions.

If you have any issues, please check your server logs and the README.md.
